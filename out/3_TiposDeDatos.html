TIPOS DE DATOS EN PYTHON

En Python, los tipos de datos son esenciales para organizar y manipular informacin. Dominarlos es crucial para escribir cdigo eficiente y preciso. En esta gua, exploraremos los tipos de datos bsicos y avanzados, sus caractersticas y ejemplos prcticos para ayudarte a dominar este tema fundamental.

Tipos de datos bsicos:

Numricos:

  • Enteros (int): Nmeros sin decimales, como 1, 2, 3. Ejemplo: edad = 25
  • Punto flotante (float): Nmeros con decimales, como 3.14, 1.5. Ejemplo: precio = 12.99
  • Complejos (complex): Nmeros con parte real e imaginaria, como 1+2j. Ejemplo: z = 1 + 2j
  • Cadenas (str): Secuencias de caracteres, como "Hola mundo", "Python 3". Ejemplo: nombre = "Juan Prez"
  • Booleanos (bool): Valores True o False que representan condiciones. Ejemplo: esta_activo = True

Tipos de datos avanzados:

  • Listas (list): Colecciones ordenadas y mutables de cualquier tipo de dato. Ejemplo: lista_nombres = ["Ana", "Pedro", "Mara"]
  • Tuplas (tuple): Colecciones ordenadas e inmutables de cualquier tipo de dato. Ejemplo: coordenadas = (10, 20, 30)
  • Diccionarios (dict): Colecciones no ordenadas que mapean claves a valores. Ejemplo: diccionario_edades = {"Ana": 25, "Pedro": 30}
  • Conjuntos (set): Colecciones no ordenadas y sin duplicados de cualquier tipo de dato. Ejemplo: conjunto_colores = {"azul", "verde", "rojo"}
  • Todos los tipos de datos son almacenados en python como objetos. Un objeto es una entidad que contiene datos y mtodos. los datos de un objeto se almacenan en atributos, mientras que los metodos son las acciones que el objeto puede realizar

En Python, if y else son dos palabras clave (que veremos ms adelante) que se usan para crear sentencias condicionales. Estas permiten ejecutar diferentes bloques de cdigo dependiendo de si una condicin se cumple o no. Por ejemplo, Imaginen que queremos saber si un nmero es mayor que 10.

numero = 15
if numero > 10:
  print("El nmero es mayor que 10")
else:
  print("El nmero no es mayor que 10")
#! nota: importante el tabulado en el if y else
